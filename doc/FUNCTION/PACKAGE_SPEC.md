# 패키지별 상세 기능명세

최종 갱신일: 2026-07-01

## `authentication`

### 목적

로컬 계정과 카카오, 구글, 네이버 OAuth 계정을 하나의 사용자 계정으로 인증한다.

### 기능

- 이메일/비밀번호 회원가입 및 로그인
- 카카오·구글·네이버 인가 코드 처리
- OAuth 공급자 식별자 기반 로그인
- 동일 사용자의 복수 OAuth 계정 연결
- 연결 대기 토큰 발급 및 기존 계정 연결
- JWT 발급, 만료 검증, 요청 사용자 ID 주입

### 주요 데이터

- `users`: 서비스 사용자
- `user_auth_identities`: 사용자별 로그인 공급자
- `oauth_pending_identities`: 계정 연결 대기 정보

## `user`

### 목적

회원 정보와 커뮤니티 프로필 관계를 제공한다.

### 기능

- 내 회원 정보 조회/수정
- 공개 프로필 조회
- 게시글 수, 팔로워 수, 팔로잉 수 집계
- 팔로우/언팔로우
- 신규 팔로워 알림 생성

## `planner`

### 목적

지역 체류 계획을 5단계로 구성하기 위한 탐색·계산 API를 제공한다.

### 생성 단계

1. 지역 선택
2. 일자리 검증 및 선택
3. 숙소 매칭과 통근 경로 확인
4. 급여·숙박비·고정비 기반 예산 계산
5. 최종 후보 저장

### 부가 기능

- 일자리 페이지 조회와 유형 필터
- TourAPI 숙박 정보 조회
- ODsay 대중교통 경로 조회
- 카카오 모빌리티 운전 경로 조회
- 지도 제공자 정보 조회

## `myplanner`

### 목적

생성 완료된 플래너를 사용자별로 저장하고 월간 일정표로 관리한다.

### 기능

- 사용자 소유 플래너 목록 조회
- 플래너 생성, 수정, 삭제
- 제목·메모·지역·재무 정보 저장
- 선택 일자리와 숙소 JSON 저장
- 일정 유형과 색상 관리
- 날짜 지정/주간 반복 일정 관리
- 플래너 생성 완료 알림
- 일정 시작 시각 알림

## `community`

### 목적

워킹홀리데이 경험을 이미지 중심 피드로 공유한다.

### 기능

- 텍스트와 다중 이미지 게시글 작성
- 본문 `#태그` 검색
- 단일 피드/3열 갤러리 조회
- 게시글 수정 및 소프트 삭제
- 좋아요/좋아요 취소
- 댓글 조회·작성·삭제
- 사용자별 피드와 내 게시글 조회
- 이벤트 태그 기반 후기 조회
- 좋아요·댓글 알림 생성

### 이미지 저장

- 로컬: `app.upload-dir`
- 운영: S3 `community/{postId}/...`

## `notification`

### 목적

플래너와 커뮤니티의 주요 변화를 최근 24시간 동안 사용자에게 전달한다.

### 알림 유형

| 유형 | 수신자 | 발생 조건 |
| --- | --- | --- |
| `PLANNER_CREATED` | 플래너 작성자 | 플래너 최초 저장 |
| `PLANNER_SCHEDULE` | 플래너 소유자 | 일정 시작 시각 도달 |
| `COMMUNITY_POST_CREATED` | 게시글 작성자 | 게시글 등록 완료 |
| `COMMUNITY_LIKE` | 게시글 작성자 | 다른 사용자가 좋아요 |
| `COMMUNITY_COMMENT` | 게시글 작성자 | 다른 사용자가 댓글 작성 |
| `NEW_FOLLOWER` | 팔로우 대상 | 다른 사용자가 팔로우 |

### 정책

- 같은 이벤트는 `dedupe_key`로 중복 생성을 방지한다.
- 본인이 자기 게시글에 남긴 반응은 알림을 생성하지 않는다.
- 생성 후 24시간이 지난 알림은 조회에서 제외하고 정리한다.
- 읽지 않은 알림이 있으면 헤더 벨에 빨간 `N`을 표시한다.
- 읽음은 개별 처리와 모두 읽음을 지원한다.
- `notifications`는 사용자·대상 FK 없이 ID 값과 인덱스로 연결한다.

## `admin`

### 목적

운영자가 서비스 현황과 사용자 생성 콘텐츠를 관리한다.

### 기능

- 일자별 회원·플래너·커뮤니티 생성 통계
- 회원 목록과 역할 변경
- 일자리·숙소 목록 및 삭제
- 플래너 현황 조회
- 커뮤니티 게시글 검색·미리보기·수정·삭제
- 외부 API 연결 상태 확인

## `festival`, `tourapi`, `region`, `job`

### 목적

플래너와 탐색 화면에 필요한 지역·관광·일자리 데이터를 제공한다.

### 기능

- 지역 메타데이터와 추천 지역 조회
- 축제, 관광지, 음식점 목록 조회
- TourAPI 공통/소개 상세 조회
- 일자리 목록 및 조건 필터

외부 API 오류나 빈 응답은 서비스가 처리 가능한 기본 응답으로 변환한다.

## `chat`

### 목적

로그인 사용자의 질문에 서비스 안내와 플래너 문맥을 반영한 답변을 제공한다.

### 기능

- OpenAI API 기반 답변
- 최신 개인 플래너의 지역·일자리·숙소·예산 문맥 반영
- 외부 AI 호출 실패 시 기본 안내 응답 제공
- 읽기 전용 도우미로 데이터 변경은 수행하지 않음

## `health`

- 애플리케이션 상태 확인
- DB 연결 상태 확인
- 배포 후 헬스 체크에 사용
