# 기능명세서

최종 갱신일: 2026-07-01

이 디렉터리는 `src/main/java/com/samteo/domain`의 패키지를 기준으로 현재 구현된 기능을 설명한다.
화면 사용 순서는 [`../MANUAL/USER_FLOW.md`](../MANUAL/USER_FLOW.md), API 목록은
[`../API/README.md`](../API/README.md)를 참조한다.

## 패키지별 기능

| 패키지 | 기능 영역 | 주요 기능 |
| --- | --- | --- |
| `admin` | 관리자 | 통계, 회원 권한, 일자리·숙소·플래너·커뮤니티 관리, API 상태 |
| `authentication` | 인증 | 로컬 회원가입/로그인, 카카오·구글·네이버 OAuth, 계정 연결, JWT |
| `chat` | 삼터 도우미 | 로그인 사용자 질문 응답, 최신 플래너 문맥 활용 |
| `community` | 커뮤니티 | 게시글·다중 이미지·태그·좋아요·댓글·사용자별 피드 |
| `festival` | 이벤트 콘텐츠 | 축제·관광지·음식점 목록 조회 |
| `health` | 상태 점검 | 서버와 DB 연결 상태 확인 |
| `job` | 일자리 | 일자리 목록과 필터 조회 |
| `myplanner` | 저장 플래너 | 개인 플래너 CRUD, 일정·유형·예산·메모 관리 |
| `notification` | 알림 | 최근 24시간 알림, 읽음/모두 읽음, 일정·커뮤니티·팔로우 알림 |
| `planner` | 플래너 생성 | 지역, 일자리, 숙소, 이동 경로, 예산 계산, 최종 플래너 생성 |
| `region` | 지역 | 지역 메타데이터 조회와 추천 |
| `tourapi` | 관광 API | 관광지·축제·상세 공통/소개 정보 중계 |
| `user` | 회원/프로필 | 내 정보, 공개 프로필, 팔로우/언팔로우, 활동 통계 |

## 공통 정책

- 인증 API와 공개 조회 API를 제외한 요청은 JWT가 필요하다.
- API 응답은 `ApiResponse<T>`의 `success`, `data`, `message` 형식을 사용한다.
- 삭제 가능한 커뮤니티 데이터는 `deleted_at`을 사용하는 소프트 삭제를 우선한다.
- 신규 서비스 테이블은 가능한 경우 FK 제약을 만들지 않고 식별자 컬럼과 인덱스로 조회한다.
- DTO는 프로젝트 규칙에 따라 Java `record` 대신 일반 클래스를 사용한다.
- 업로드 이미지는 로컬에서는 임시 업로드 경로, 운영에서는 S3를 사용한다.

상세 명세는 [`PACKAGE_SPEC.md`](PACKAGE_SPEC.md)를 참조한다.
