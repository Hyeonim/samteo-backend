# 사용자 사용 흐름서

최종 갱신일: 2026-07-01

## 1. 서비스 진입과 로그인

1. 메인 페이지에서 삼터가 제공하는 쉼터·일터·놀터 기능을 확인한다.
2. 헤더의 `로그인 / 회원가입`을 선택한다.
3. 이메일 계정 또는 카카오·구글·네이버 계정으로 로그인한다.
4. JWT가 만료되면 재로그인 안내가 표시되며 보호 화면은 다시 인증해야 한다.

## 2. 플래너 만들기

1. 로그인 후 `내 플래너 시작하기` 또는 플래너 생성 버튼을 선택한다.
2. 최초 진입 시 체류 유형과 단계별 온보딩 가이드를 확인한다.
3. `지역 선택`에서 체류할 지역을 고른다.
4. `일자리 검증`에서 공고를 검색·필터하고 후보를 선택한다.
5. `숙소 매칭`에서 숙소와 통근 경로를 비교한다.
6. `가계부 연산`에서 급여, 숙박비, 생활비와 예상 잔액을 확인한다.
7. `플래너 완성`에서 후보를 저장하고 내 플래너로 이동한다.

## 3. 내 플래너 관리

1. 헤더 `내 플래너`에서 저장된 플래너를 선택한다.
2. 제목과 메모를 수정한다.
3. 일간·주간·월간 일정표를 전환한다.
4. 날짜를 선택해 개인 일정의 제목, 시간, 유형, 색상을 등록한다.
5. 플래너 생성 시 완료 알림, 일정 시간이 되면 시작 알림을 받는다.

## 4. 일자리·이벤트 탐색

- `일자리`: 추천 조건, 마감 임박, 인기 공고를 조회하고 숙소 매칭을 시작한다.
- `이벤트`: 축제·관광지·음식점을 지역별로 확인하고 플래너에 담는다.
- 이벤트 후기 버튼에서 해당 이벤트 태그가 포함된 커뮤니티 글을 3열로 확인한다.
- 후기 글의 작성자 프로필, 좋아요, 댓글 기능을 그대로 사용할 수 있다.

## 5. 커뮤니티

1. 헤더 `커뮤니티`로 이동한다.
2. 단일 피드 또는 3열 갤러리 보기 방식을 선택한다.
3. `게시글 만들기`에서 이미지를 여러 장 추가하고 본문을 작성한다.
4. 공백으로 구분된 `#태그`는 태그 링크로 표시된다.
5. 미리보기에서 이미지 이동, 줄바꿈, 태그 표시를 확인하고 게시한다.
6. 다른 사용자의 게시글에 좋아요나 댓글을 남긴다.
7. 작성자 프로필로 이동해 피드와 팔로워/팔로잉 정보를 보고 팔로우한다.

## 6. 알림

1. 로그인 상태에서 헤더 내 정보 왼쪽의 벨을 확인한다.
2. 미확인 알림이 있으면 빨간 `N`이 표시된다.
3. 벨을 누르면 최근 24시간 알림과 `몇 분 전`, `몇 시간 전` 시간이 표시된다.
4. 연두색 항목은 미확인, 흰색 항목은 읽은 알림이다.
5. 항목을 누르면 읽음 처리 후 관련 화면으로 이동한다.
6. `모두 읽음`으로 전체 미확인 알림을 한 번에 처리할 수 있다.

## 7. 마이페이지

- 회원명, 이메일, 연결된 로그인 공급자를 확인한다.
- 내 게시글을 미리보고 수정하거나 삭제한다.
- 게시글 수, 팔로워 수, 팔로잉 수를 확인한다.

## 8. 삼터 도우미

- 화면 우측 하단의 도우미 버튼을 연다.
- 서비스 사용법이나 현재 플래너와 관련된 질문을 입력한다.
- 도우미는 조회와 안내만 제공하며 사용자 데이터를 직접 수정하지 않는다.
