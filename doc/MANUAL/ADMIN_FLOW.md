# 관리자 사용 흐름서

최종 갱신일: 2026-07-01

관리자 계정은 헤더의 `관리자` 메뉴로 진입한다.

## 대시보드

1. 조회 기간을 선택한다.
2. 회원 가입, 플래너 생성, 커뮤니티 게시글 생성 수를 확인한다.
3. 일자별 막대그래프로 변화 추이를 비교한다.

## 회원 관리

1. 회원 목록과 이메일, 가입 방식을 확인한다.
2. 일반 사용자와 관리자 역할을 변경한다.
3. 현재 로그인한 관리자 자신의 역할은 변경할 수 없다.

## 콘텐츠 관리

- 공고 관리: 일자리 목록을 조회하고 부적절한 공고를 삭제한다.
- 숙소 관리: 숙소 목록을 조회하고 잘못된 숙소를 삭제한다.
- 플래너 현황: 회원별 저장 플래너, 지역, 유형, 예산을 조회한다.
- 커뮤니티 관리: 게시글 검색, 이미지 미리보기, 본문 수정, 게시글 삭제를 수행한다.

## API 상태

외부 관광·교통·AI API 상태를 확인하고 장애 시 운영 환경변수와 서버 로그를 점검한다.
